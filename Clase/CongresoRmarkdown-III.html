DPLYR

Dplyr es un paquete que permite la manipulación de datos, proporciona un conjunto de funciones que ayudan a resolver los desafíos más comunes de manipulación de datos:

  • mutate() : agrega nuevas variables que son funciones de variables existentes.
  • select() : selecciona variables basadas en sus nombres.
  • filter() : elige casos en función de sus valores.
  • summarise() : reduce múltiples valores hasta un solo resumen.
  • arrange() : cambia el orden de las columnas.

GGPLOT2

library(gapminder)
library(dplyr)
library(ggplot2)

gapminder_2007  <-  gapminder   %>%
        filter(year ==  2007)

# Scatter Plot
ggplot(gapminder_2007,  aes(x   =   gdpPercap,  y   =   lifeExp))   +
        geom_point()

# Escala logarítimica
ggplot(gapminder_2007,  aes(x   =   gdpPercap,  y   =   lifeExp))   +
        geom_point()    +
        scale_x_log10()

Estética (aes)

# Color
ggplot(gapminder_2007,  aes(x   =   gdpPercap,  y   =   lifeExp,    color   =   continent)) +
        geom_point()    +
        scale_x_log10()

# Tamaño
ggplot(gapminder_2007,  aes(x   =   gdpPercap,  y   =   lifeExp,    color   =   continent, size =   pop))   +
        geom_point()    +
        scale_x_log10()

# Agrupado
ggplot(gapminder_2007,  aes(x   =   gdpPercap,  y   =   lifeExp))   +
        geom_point()    +
        scale_x_log10() +
        facet_wrap(~continent)
